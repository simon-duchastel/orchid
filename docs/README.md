# Orchid Documentation

Welcome to the Orchid documentation. This documentation provides comprehensive guides for understanding, developing, and contributing to the Orchid project.

## What is Orchid?

Orchid is a CLI tool that orchestrates AI agents in the background. It manages a daemon process that runs an OpenCode server, allowing you to execute complex multi-agent AI tasks without blocking your terminal.

## Quick Start

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Link for global CLI usage
npm link

# Start the daemon
orchid up
```
